Traverse a tree (DFS & BFS)

Date: November 2, 2016
Category: Data Structure

前言

紀錄 Tree 的用途還有 DFS 與 BFS 的概念

Tree 的用途

樹狀結構可以用在很多地方

Syntax Tree for sentence

I ate the cake

Expression

sin(3z - 7)

Abstract Syntax Tree

Binary Search Tree (BST)

通常是左小又大，這邊放的例子是以字母(ASCII)排序

Tree 的結構

一棵樹有好幾個節點，其中的關係有

  • Root (最頂端的節點)
  • Ancestor (父母、祖先)
  • Decendants (小孩、子孫)
  • Sibling (兄弟)
  • Leaf (no children)

一棵樹的高度定義為

Height: maximum depth of subtree node and farthest leaf

可以用 recursive 來計算樹的高度

拜訪一棵樹有兩種做法

  1. Depth First
  2. Breadth First

Depth First

先找leaf，再找sibling

DFS 基本上就是 recursive

Inorder 中序

left → root → right

Pre-order 前序

root → left → right

Post-order 後序

left → right → root

Breadth First

level by level

先把sibling找完了才往下找

通常實作 BFS 會用 Queue 的特性來實現
