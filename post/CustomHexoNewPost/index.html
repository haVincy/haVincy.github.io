自訂 Hexo new post 的預設內容

Date: January 17, 2020
Category: Blogging

前言

在改動了一些 Hexo 的設定後，每次 New post 時的預設模板，對我來說已經不太方便

上方的 Front-Matter 的欄位太少了，需要自己新增欄位然後給值，很煩人，所以就研究了下該如何改變他的預設模板

New Post

找到 post.md

在裡面新增想要的欄位就可以了，使用 Hexo new post 時上方就會出現你預先寫好的欄位了

也可以自己新增一些每次新增文章都會有的大綱模板，這樣就不需要每次都再打一次了！
