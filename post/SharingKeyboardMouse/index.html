Sharing input devices, 在不同電腦上共享鍵盤滑鼠

Date: January 9, 2020
Category: Tips

前言

因為工作需要的關係，之前需要同時在 Ubuntu (Linux) 和 Windows 上切換，一直深感困擾

後來換要開發 iOS，用公司的 Mac 開發偶爾也還是需要回到 Ubuntu 去 build 一些東西，後來才知道同事們用各種不同的方式，讓自己能在不同電腦的螢幕之間切換，但不需要換鍵盤跟滑鼠(大家都好有效率啊)

為了提升效率，我選擇用 Synergy 這款軟體，可以讓我鍵盤與滑鼠接在 Ubuntu 的主機上，讓我的 Mac 連接到 Ubuntu 的主機，使用同一組鍵盤滑鼠

Synergy

強者同事說官網的軟體需要註冊然後據說要付費

可以到這邊下載還沒付費之前被人 clone 出來的 synergy source

https://github.com/brahma-dev/synergy-stable-builds/releases

根據你要共享的作業系統下載對應的執行檔

設定上很簡單，分為 Server side & Client side

需要注意的是，兩台電腦需要連在共同的網域下才能互連

Server side

要有固定 ip 才能夠讓 client 連線進來，所以一般家用主機可能比較難有固定 ip，除非有跟網路公司申請

打開 Synergy 勾選 Server 那一欄，輸入 IP address，然後設定一下想要的螢幕分享操作模式，按下 Apply

Client side

勾選 Client 那一欄，輸入要連接的 Server IP，按下 Apply

兩台電腦就會開始接收傳送訊號了，如下圖

缺點就是需要一直開著軟體(廢話)，不過還是比要切來切去好多了，Synergy 會記住你上一次的設定所以也不用怕關掉重開要重設東西
