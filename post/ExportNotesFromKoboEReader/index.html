輸出 Kobo 電子閱讀器的註記

Date: March 17, 2020
Category: Tips

為什麼買電子閱讀器

最重要的原因當然就是攜帶方便，對租屋族來說不會堆書在家，還有護眼，再來就是個人覺得比較好做筆記，今年的一個小目標是想把看完的書裡面的畫記都記錄下來，而不是每次都只有在看完的當週記憶力最好，以為自己又漲知識，過了一個月只記得零碎的內容，實際上吸收效率很差

買實體書用筆做筆記然後要記錄到電腦裡又太耗時，用電子書的方式是最方便的，記完後直接輸出就好了

為什麼選 Kobo

去年趁著雙十一購物節購入了人生第一台電子書閱讀器，挑選的是 Kobo Libra H2O，整體使用起來很滿意，大小剛好，單手也很可以

當初上網 survey 了一下，一直在 Amazon 的 Kindle 與 Kobo 之間抉擇

最終選擇 Kobo 的原因除了完善的中文支援，主要是註記的功能， Kindle 雖然註記比較方便，介面經過這些年的演進越來越好，但有限制匯出的註記不能超過全書的 10%，對於我個人使用習慣而言就很不方便，加上 Kobo 的價錢確實比較甜XD

中文書與英文書比例，Kobo 的樂天能支援的數量是屬於中間，而英文書最新最多的就是 Amazon，如果比較偏好看英文書，買 Kindle 搭配 Amazon 的商店來說或許會是最好的，主要的閱讀書群是中文類但也不排除會看英文書的話，我相信 Kobo 會是兩者兼具的好選擇

閱讀器 vs 平板

電子書閱讀器，又叫 E-Reader、E-Ink、電子紙、電子墨，是為需要長時間閱讀螢幕者所設計的電子載體

所謂的電子墨水是由數百萬個微膠囊所構成，在每個微膠囊裡均含有帶負電荷的白色電泳粒子，以及帶正電荷的黑色電泳粒子，把電子墨水塗佈在一層塑膠薄膜上，再貼覆上薄膜電晶體（TFT）電路，利用正負相吸的原理，當電場接通時，該區塊對應的黑或白粒子會移動至微膠囊的頂端，使用者在該區塊上，就能看見白色或黑色，經由驅動 IC 控制，便可形成像素圖形，在電子紙螢幕上顯現出來。

平板拿來閱讀第一個要考慮的就是眼睛健康，因為平板的螢幕是 LED 自體發光，而電子墨的螢幕是依賴環境光源，雖然現在 Android/iOS 的軟體都有針對環境光線自己調整螢幕光源，但對人眼來說刺激還是比較大

電子閱讀書也比一般螢幕來得省電，因為他只會在畫面變化時才會需要電力，單純顯示上的耗電率很低

如果是為了搭配上課筆記的話，買平板會是比較好的選擇，大二那時候終於受不了要印一堆紙本上課，買了iPad (第三代)，使用筆記類的 APP 輸入上課用的 PPT 或教材，可以在上面做各種筆記跟新增一個空白頁，對討厭帶一堆紙的我而言簡直是救星

以單純筆記跟學習而言，那就買平板，若是單純為了閱讀，那我會認為買電子閱讀器是最好的

Kobo 匯出註記

進入正題，前面提到我買電子閱讀器的動機之一是為了能方便整理畫在書中的註記，而 Kobo default 居然沒有選項能夠匯出註記，上網查到解法是需要自己去改 Kobo 的設定

改動設定檔

  1. 將 Kobo 電子閱讀器連接到電腦
  2. 打開 Kobo eReader 的 folder
  3. 找到 .kobo -> Kobo -> Kobo eReader.conf

用文字編輯器打開 Kobo eReader.conf

如果電腦有打 code 常用的幾個文字編輯器就選一個自己喜歡用的就好了，若是沒有，在 Windows 右鍵用管理員身份用記事本開啟也可以，Mac 的話用文字編輯打開

匯出註記

安全移除你的 Kobo eReader，這時候進到你的書籍，點擊在某本書籍旁邊的 More info (三個點點)，就會看到匯出畫線的選項了

選擇匯出畫線後，他會跳出一個視窗，會輸出成 .txt 檔案在母資料夾下，可以連接電腦 check 看看

所以實際上 Kobo eReader 已經支援匯出註記，但卻不公開給使用者使用XD，否則我們也無法只改設定檔就新增一個項目在選單
